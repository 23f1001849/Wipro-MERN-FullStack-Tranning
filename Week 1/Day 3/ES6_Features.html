Spread & object updates

Spread lets you clone arrays or objects and add new values without mutating the original.

const numbers = [1, 2, 3];
const extended = [...numbers, 4, 5];

const person = { name: "Alice", age: 25 };
const withCity = { ...person, city: "New York" };

console.log(extended);
console.log(withCity);

Arrow functions & defaults

Arrow functions keep callbacks concise and pair nicely with default parameters.

const add = (a, b = 0) => a + b;
const facts = ["fast", "portable", "expressive"];

const summary = facts.map((fact, index) =>
  `${index + 1}. ES6 is ${fact}`
);

console.log(add(4, 6));
console.log(summary.join("\n"));

Classes & inheritance

ES6 classes encapsulate state and behaviour; extends helps you reuse logic smoothly.

class Person {
  constructor(name, role) {
    this.name = name;
    this.role = role;
  }
  describe() {
    return `${this.name} works as ${this.role}.`;
  }
}

class Mentor extends Person {
  constructor(name, role, topics) {
    super(name, role);
    this.topics = topics;
  }
  intro() {
    return `${this.describe()} Teaches ${this.topics.join(", ")}.`;
  }
}

Map & Set collections

Maps store keyed data, Sets keep unique values—ideal for caching, tagging, and membership checks.

const languages = new Map([
  ["JS", "JavaScript"],
  ["TS", "TypeScript"],
]);

const attendees = new Set(["Alice", "Bob", "Alice"]);

console.log([...languages.entries()]);
console.log(attendees.size);
console.log(attendees.has("Bob"));
