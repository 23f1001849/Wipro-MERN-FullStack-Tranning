Block scope & string templates

Work with let/const, template literals, and object destructuring without leaking variables.

const pi = Math.PI;
const radius = 5;
const area = (pi * radius ** 2).toFixed(2);

const user = { firstName: "John", lastName: "Doe" };
const { firstName, lastName } = user;

console.log(`Area: ${area}`);
console.log(`Name: ${firstName} ${lastName}`);

Arrow functions & spread syntax

Arrow functions pair well with default parameters and the spread operator for concise data work.

const square = (x) => x * x;
const multiply = (a, b = 2) => a * b;

const numbers = [1, 2, 3];
const more = [...numbers, 4, 5];

console.log(square(4));
console.log(multiply(5));
console.log(more);

Classes & inheritance

ES6 classes wrap constructors and methods in a friendly syntax, including inheritance via extends.

class Animal {
  constructor(name) {
    this.name = name;
  }
  speak() {
    return `${this.name} makes a noise.`;
  }
}

class Dog extends Animal {
  speak() {
    return `${this.name} barks.`;
  }
}

new Dog("Rex").speak();

Map & Set collections

Leverage Map for key-value lookups and Set for unique lists without manual deduping.

const capitals = new Map([
  ["USA", "Washington, D.C."],
  ["Canada", "Ottawa"],
]);

const visited = new Set(["Tokyo", "Berlin", "Tokyo"]);

console.log([...capitals.entries()]);
console.log([...visited]);
