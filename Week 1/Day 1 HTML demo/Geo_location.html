About This Demo

Semantic HTML works hand-in-hand with the Geolocation API. When you click “Use My Location” we request coordinates, calculate the straight-line distance to Delhi, and update the map marker.

All time data is generated on the fly using Intl.DateTimeFormat with the Asia/Kolkata timezone. The map uses an embedded OpenStreetMap frame. Nothing is stored or transmitted.
