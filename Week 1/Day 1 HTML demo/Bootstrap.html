Bootstrap Quickstart

Drop-in CSS and JavaScript utilities, components, and layout helpers for building modern, responsive sites faster. Include the CDN link above and start composing interfaces with ready-made building blocks.

Layout Utilities

Use .container, .row, and .col to create fluid grids that adapt to every breakpoint.

  • Flexbox helpers .d-flex, .justify-content-*
  • Spacing system via .p-*, .m-*
  • Responsive display toggles like .d-md-none

Components

Drop in buttons, alerts, cards, modals, accordions, navbars, and dozens of other patterns.

  • Utility classes stack with component classes
  • Customizable via SCSS variables or CSS overrides
  • Accessible by default with ARIA attributes baked in

Getting Started

  1. Add the CDN CSS inside <head>
  2. Compose UI with Bootstrap classes
  3. Optionally include the bundle JS for interactivity
  4. Test at sm, md, lg viewports
