# Day 16 · Node.js Basics Server

Simple HTTP server (no frameworks) used to explain fundamental Node topics.

## Scripts

```bash
npm install   # only needed if you add deps later
npm run dev   # starts the server on port 3016
```

## Routes

- `/intro` – What Node is, history, features, advantages.
- `/install` – Installation tips + detects OS info.
- `/npm` – npm description and common commands.
- `/sample-app.js` – Minimal Node HTTP snippet to show students.
