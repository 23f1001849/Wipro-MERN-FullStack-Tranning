# Day 19 · Express.js Introduction

Covers what Express is, why to use it, and how to bootstrap a project.

## Run

```bash
npm install
npm run dev
```

## Routes

- `/` – Overview of Express purpose + features.
- `/getting-started` – Step list plus sample snippet.
- `/echo` (POST) – Demonstrates JSON parsing.
- `/health` – Simple heartbeat endpoint for uptime checks.
